## 记号约定

在块级引用中的部分，均为定理或者定义

例：

> **约定定理:** 这里是一个定义

> **约定定义:** 这里是一个定义

斜体字出现的部分，表示将使用不严谨的语言进行类比或者牢骚，仅供参考
例：
*这里是一段牢骚*
